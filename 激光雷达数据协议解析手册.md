## 2. 字段详细描述

|名称|长度|描述|
|---|---|---|
|PH(2B)|数据包头|长度为 2B，固定为`0x55AA`，低位在前，高位在后|
|CT(1B)|数据类型|`CT[bit(0:1)]`表示为一帧数据起始：
- `01`：点云数据包
- `11`：预留位|
|LSN(1B)|采样数量|表示当前数据包中包含的采样点数量（整数），数据包中只有 1 个起始点的数值|
|FSA(2B)|起始角|采样数据中第一个采样点对应的角度数据|
|LSA(2B)|结束角|采样数据中最后一个采样点对应的角度数据|
|CS(2B)|校验码|当前数据包的校验码，采用双字节异或对当前数据包进行校验|
|S(3B)|采样数据|系统测试的采样数据，为点云的距离数据，其中 S1 点的 LSB 中还集成了点信息|

## 3. 起始位与扫描频率解析

- 当检测到`CT[bit(0)]=0`时，表明该数据包为**点云数据包**

- 当检测到`CT[bit(0)]=1`时，表明该数据包为**起始数据包**，该数据包中`LSN=1`，即 Si 的数量为 1；其距离、角度的具体值解析参见下文；同时，起始数据包中，`CT[bit(7:1)]`扫描频率信息，`F = CT[bit(7:1)] / 10`（当`CT[bit(7:1)]=1`时）

- 注：当`CT[bit(7:1)]=0`时，`CT[bit(7:1)]`为预留位，未来版本会用作其他用途；因此在解析 CT 过程中，只需要对`bit(0)`位做起始帧的判断

## 4. 距离解析

### 计算公式

```Plain Text

Distance_i = s_i / 4
```

其中：

- `s_i` 为采样数据（小端模式存储）

- 结果单位：毫米 (mm)

### 计算示例

某采样数据为`E5 0F`，由于本系统是小端模式，所以采样点 S = `0x0FE5`
带入距离计算公式得：

```Plain Text

Distance = 0x0FE5 / 4 = 4069 / 4 = 1017.25 mm
```

## 5. 角度解析

角度数据保存在 FSA 和 LSA 中，每个角度数据有如下的数据结构：C 是校验位，其值固定为 1。角度解析有两个等级：一级解析和二级解析。一级解析初步得到角度初值，二级解析对角度初值进行修正。

### 5.1 一级解析（角度初值）

- 起始角计算公式：

    ```Plain Text
    
    Angle_FSA = Rbti(fbit(FSA, 1)) / 64
    ```

- 结束角计算公式：

    ```Plain Text
    
    Angle_LSA = Rbti(fbit(LSA, 1)) / 64
    ```

- 中间角计算公式：

    ```Plain Text
    
    Angle_i = (diff(Angle) / (LSN - 1)) * (i - 1) + Angle_FSA
    (i = 2, 3, ..., LSN - 1)
    ```

**符号说明**：

- `Rbti(fbit(data, 1))`：表示将数据 data 右移一位

- `diff(Angle)`：表示起始角（未修正值）到结束角（未修正值）的顺时针角度差

- `LSN`：表示本帧数据包采样数量

### 5.2 二级解析（角度修正）

#### 角度修正公式

```Plain Text

Angle_i = Angle_i + AngCorrect_i  (i = 1, 2, ..., LSN)
```

其中，`AngCorrect_i`为角度修正值，其计算公式如下（`tan⁻¹`为反三角函数，返回角度值）：

```Plain Text

IF Distance_i == 0
    AngCorrect_i = 0
ELSE
    AngCorrect_i = tan⁻¹( (21.8 * (155.3 - Distance_i)) / (155.3 * Distance_i) )
```

### 5.3 完整计算示例

设数据包中，第 4~8 字节为`E5 6F BD 79 79`，所以：

- `LSN = 0x28 = 40`（dec）

- `FSA = 0x6FE5`

- `LSA = 0x79BD`

#### 第一步：一级解析

```Plain Text

Angle_FSA = Rbti(0x6FE5) / 64 = 0x37F2 / 64 = 14322 / 64 = 223.78125°
Angle_LSA = Rbti(0x79BD) / 64 = 0x3CDE / 64 = 15582 / 64 = 243.46875°
diff(Angle) = 243.46875° - 223.78125° = 19.6875°
```

中间角公式：

```Plain Text

Angle_i = (19.6875° / 39) * (i - 1) + 223.78125°  (i = 2, 3, ..., 39)
```

#### 第二步：二级解析

假设该帧数据中：

- `Distance₁ = 1000 mm`

- `Distance_LSN = 8000 mm`

带入二级解算公式：

```Plain Text

AngCorrect₁ = tan⁻¹( (21.8 * (155.3 - 1000)) / (155.3 * 1000) )
            = tan⁻¹( (21.8 * (-844.7)) / 155300 )
            = tan⁻¹( -18414.46 / 155300 )
            = tan⁻¹( -0.11857 )
            = -6.7622°

AngCorrect_LSN = tan⁻¹( (21.8 * (155.3 - 8000)) / (155.3 * 8000) )
               = tan⁻¹( (21.8 * (-7844.7)) / 1242400 )
               = tan⁻¹( -171014.46 / 1242400 )
               = tan⁻¹( -0.13765 )
               = -7.8374°
```

修正后最终角度：

```Plain Text

Angle_FSA = 223.78125° + (-6.7622°) = 217.01905°
Angle_LSA = 243.46875° + (-7.8374°) = 235.63135°
```

同理，`Angle_i (i = 2, 3, ..., LSN - 1)` 可以依次求出。
