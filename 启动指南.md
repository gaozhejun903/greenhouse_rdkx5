# LiDAR 启动指南

## 1. 前置条件

确保已编译工作空间：

```bash
cd /root/lidar-check_ws
colcon build
```

## 2. 启动方式

共有 3 种启动模式，按需选择：

### 2.1 仅启动雷达驱动

```bash
source /root/lidar-check_ws/install/setup.bash
ros2 launch lidar_pkg lidar.launch.py
```

只启动 LiDAR 节点，发布 `/scan` 话题。

### 2.2 启动雷达 + RViz 可视化

```bash
source /root/lidar-check_ws/install/setup.bash
ros2 launch lidar_pkg monitor.launch.py
```

启动 LiDAR 节点 + TF 坐标变换 + URDF 模型 + RViz 可视化。

### 2.3 启动雷达 + SLAM 建图

```bash
source /root/lidar-check_ws/install/setup.bash
ros2 launch lidar_pkg slam.launch.py
```

启动 LiDAR 节点 + TF + URDF + slam_toolbox 异步 SLAM + RViz 可视化。

## 3. 查看数据

在新终端中：

```bash
source /root/lidar-check_ws/install/setup.bash

# 查看雷达话题
ros2 topic list
ros2 topic echo /scan
ros2 topic hz /scan
```

## 4. 关闭

按 `Ctrl+C` 终止，或：

```bash
pkill -f lidar_node
```
