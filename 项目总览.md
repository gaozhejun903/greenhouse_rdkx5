## 一、项目简介

基于 **RDK X5** 的大棚作物病虫害检测机器人系统。机器人搭载激光雷达实现 SLAM 建图与自主导航定位，搭载深度相机进行病虫害视觉检测，检测到病虫害后记录其地图坐标，最终通过局域网将数据上传至电脑端。

### 核心功能

1. **SLAM 建图** — 激光雷达构建大棚 2D 地图
2. **自主导航定位** — Nav2 导航栈，小车在地图中自主移动 + 实时定位
3. **病虫害检测** — 深度相机 + 视觉模型，检测病虫害种类
4. **病虫害坐标标记** — 检测到病虫害时，记录当前小车在地图中的坐标
5. **数据上传** — 通过局域网将病虫害数据（种类 + 坐标）上传至电脑端 Python 接收

## 三、通信协议

### 3.1 RDK X5 → STM32 控制协议

格式：`AA + CMD + 55`（3 字节）

| 指令 | 字节 | 说明 |
|------|------|------|
| 前进 | `AA 11 55` | 小车前进 |
| 后退 | `AA 12 55` | 小车后退 |
| 停止 | `AA 13 55` | 小车停止（注：协议为 AA 13 55） |
| 前左转 | `AA 14 55` | 小车向前左转 |
| 前右转 | `AA 15 55` | 小车向前右转 |

> **注意**：当前 `base_driver.cpp` 中的 `cmd_byte` 映射为 `0x00~0x04`，需要与 STM32 固件中的实际字节对应。如果 STM32 使用 `0x11~0x15`，需要修改映射。

### 3.2 病虫害检测返回协议

格式：`AA + TYPE + 55`（3 字节）

| 字节 | 含义 |
|------|------|
| `AA 00 55` | 病虫害类型 0（待定义具体种类） |
| `AA 01 55` | 病虫害类型 1（待定义具体种类） |
| `AA 02 55` | 病虫害类型 2（待定义具体种类） |
| ... | 更多类型待视觉模型确定后补充 |

> 视觉模型检测到病虫害后，返回对应字节，系统同时记录当前小车在地图中的坐标。

### 3.3 数据上传协议（RDK X5 → 电脑端 Python）

通过局域网 TCP/UDP 上传，数据格式（待实现）：

```json
{
  "type": "pest_report",
  "pest_type": 0,
  "pest_type_name": "蚜虫",
  "map_x": 1.23,
  "map_y": 4.56,
  "map_theta": 0.78,
  "timestamp": "2026-06-18T10:30:00",
  "image_path": "/path/to/snapshot.jpg"
}
```

## 五、现有代码说明

### 5.1 lidar_node.cpp — 激光雷达驱动

| 项目 | 内容 |
|------|------|
| **文件** | `src/lidar_pkg/src/lidar_node.cpp` |
| **功能** | 解析激光雷达 0xAA55 协议数据，发布 `/scan` 话题 |
| **协议** | PH(2) + CT(1) + LSN(1) + FSA(2) + LSA(2) + Samples(3*LSN) |
| **滤波** | 基于半径的离群点滤波（默认 0.10m 半径，最少 2 个邻居） |
| **话题** | 发布 `/scan` (sensor_msgs/LaserScan) |
| **参数** | port_name, frame_id, filter.enabled, filter.radius, filter.min_neighbors |
| **看门狗** | 2 秒无数据自动重发启动命令 |

### 5.2 base_driver.cpp — STM32 底盘驱动

| 项目 | 内容 |
|------|------|
| **文件** | `src/lidar_pkg/src/base_driver.cpp` |
| **功能** | 串口通信控制 STM32 底盘 + 航位推算里程计 |
| **协议** | `AA + CMD + 55`（3 字节） |
| **订阅** | `/cmd_vel` (Twist) — Nav2 导航速度指令 |
| **订阅** | `/cmd_byte` (UInt8) — 直接字节控制 |
| **发布** | `/odom` (Odometry) + TF `odom → base_link` |
| **安全** | 0.5 秒无指令自动停止 |
| **参数** | port, baudrate, odom_frame, base_frame, max_linear_speed, max_angular_speed, cmd_timeout |

#### cmd_byte 映射（当前代码）

| cmd_byte 值 | STM32 指令 | 动作 |
|-------------|-----------|------|
| 0x11 | AA 11 55 | 前进 |
| 0x12 | AA 12 55 | 后退 |
| 0x13 | AA 13 55 | 停止 |
| 0x14 | AA 14 55 | 前左转 |
| 0x15 | AA 15 55 | 前右转 |

### 5.3 SLAM 建图

| 项目 | 内容 |
|------|------|
| **启动文件** | `launch/slam.launch.py` |
| **算法** | slam_toolbox (async_slam_toolbox_node) |
| **参数文件** | `config/slam_params.yaml` |
| **输入** | `/scan` |
| **输出** | `/map` (OccupancyGrid) + TF `map → odom` |
| **建图后保存** | `ros2 run nav2_map_server map_saver -f maps/my_map` |

### 5.4 Nav2 导航

| 项目 | 内容 |
|------|------|
| **启动文件** | `launch/nav.launch.py` |
| **参数文件** | `config/nav2_params.yaml` |
| **地图路径** | `/root/rdkx5_ws/maps/my_map.yaml` |
| **定位** | AMCL (粒子滤波) |
| **全局规划** | NavfnPlanner (Dijkstra) |
| **局部控制** | DWB Local Planner |
| **行为树** | navigate_to_pose_w_replanning_and_recovery |
| **代价地图** | 全局(static_layer + inflation) + 局部(voxel_layer + inflation) |
| **机器人半径** | 0.15m |
| **最大速度** | 线速度 0.5m/s，角速度 1.0rad/s |

### 5.5 病虫害定位监控

| 项目 | 内容 |
|------|------|
| **文件** | `src/lidar_pkg/src/pest_monitor.cpp` |
| **功能** | 订阅 AMCL 位姿 + 病虫害检测消息，记录病虫害地图坐标，TCP 上传 |
| **启动文件** | `launch/pest_monitor.launch.py` (或随 `nav.launch.py` 自动启动) |
| **参数文件** | `config/pest_monitor_params.yaml` |
| **订阅** | `/amcl_pose`, `/pest_detected` (PestDetection) |
| **发布** | `/pest_report` (PestReport), `/pest_markers` (MarkerArray), `/robot_pose` |
| **TCP 上传** | JSON 格式，自动重连，默认 192.168.1.100:8888 |

## 七、常用命令

### 编译

```bash
cd ~/rdkx5_ws
colcon build --packages-select lidar_pkg
source install/setup.bash
```

### SLAM 建图

```bash
ros2 launch lidar_pkg slam.launch.py
# 建图完成后另开终端保存
ros2 run nav2_map_server map_saver -f maps/my_map
```

### Nav2 导航

```bash
ros2 launch lidar_pkg nav.launch.py
```

### 手动控制小车

```bash
# 通过 cmd_byte 控制 (0x11~0x15)
ros2 topic pub /cmd_byte std_msgs/UInt8 "data: 17"   # 前进 (0x11)
ros2 topic pub /cmd_byte std_msgs/UInt8 "data: 18"   # 后退 (0x12)
ros2 topic pub /cmd_byte std_msgs/UInt8 "data: 19"   # 停止 (0x13)
ros2 topic pub /cmd_byte std_msgs/UInt8 "data: 20"   # 前左转 (0x14)
ros2 topic pub /cmd_byte std_msgs/UInt8 "data: 21"   # 前右转 (0x15)
```

### 查看话题

```bash
ros2 topic echo /scan         # 激光雷达数据
ros2 topic echo /odom         # 里程计数据
ros2 topic echo /amcl_pose    # 当前地图定位
ros2 topic echo /robot_pose   # 小车当前位姿 (pest_monitor 发布)
ros2 topic echo /pest_report  # 病虫害报告
```

### 模拟病虫害检测 (测试用)

```bash
# 发送一条病虫害检测消息 (类型=0, 置信度=0.95)
ros2 topic pub /pest_detected lidar_pkg/msg/PestDetection \
  "{pest_type: 0, pest_type_name: '蚜虫', confidence: 0.95, timestamp: '', image_path: ''}" --once
```
