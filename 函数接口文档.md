# 函数接口文档

> 所有函数接口均来源于实际代码，修改代码前先查阅此文档，禁止凭空捏造接口。

---

## LidarNode 类

### 公开方法 (Public)

| 函数 | 说明 |
|------|------|
| `LidarNode()` | 构造函数：声明参数、创建 publisher、打开串口、初始化雷达、启动看门狗定时器 |
| `~LidarNode()` | 析构函数：关闭雷达 |
| `void run_loop()` | 主循环：读取串口数据并逐字节解析，500Hz 频率运行 |

### 私有方法 (Private)

#### ROS 相关

| 函数 | 说明 |
|------|------|
| `void publishScan()` | 将 `full_scan_buffer_` 发布为 `sensor_msgs::msg::LaserScan` 话题 `/scan` |
| `void watchdogCallback()` | 看门狗回调：若超时未收到有效数据，重新发送启动命令 |

#### 串口相关

| 函数 | 说明 |
|------|------|
| `bool openSerial(const std::string& port, int baudrate)` | 打开并配置串口 (termios2, 150000 baud) |
| `void sendCmd(const std::vector<uint8_t>& cmd)` | 发送原始命令到串口 |
| `void flushSerial()` | 刷新串口缓冲区 |
| `void shutdown()` | 发送停止命令并关闭串口 |
| `void initLidar()` | 初始化雷达流程：stop → flush → start → flush |

#### 协议解析

| 函数 | 说明 |
|------|------|
| `uint16_t bytesToUint16(const uint8_t* data)` | 将小端序 2 字节转为 uint16（指针版本） |
| `void processByte(uint8_t byte)` | 状态机逐字节解析 (WAIT_HEADER1 → WAIT_HEADER2 → READ_META → READ_PAYLOAD)，含 LSN 大小保护 |
| `void parsePacket(const std::vector<uint8_t>& packet_data)` | 解析完整数据包：角度插值、存入缓冲区、检测旋转周期。采样偏移量 = 8 + i*3 |

#### 滤波

| 函数 | 说明 |
|------|------|
| `void removeOutliers(std::vector<LidarPoint>& points)` | 半径离群点滤波 (radius=0.10m, min_neighbors=2) |

## 数据结构

| 结构体 | 字段 | 说明 |
|--------|------|------|
| `LidarPoint` | `angle_rad` | 角度 (弧度) |
| | `distance_m` | 距离 (米) |
| | `x, y` | 笛卡尔坐标 (米) |

## 枚举

| 枚举 | 值 | 说明 |
|------|-----|------|
| `State` | `WAIT_HEADER1` | 等待 0xAA |
| | `WAIT_HEADER2` | 等待 0x55 |
| | `READ_META` | 读取 CT+LSN+FSA+LSA |
| | `READ_PAYLOAD` | 读取采样点数据 |

---

## 宏定义

| 宏 | 值 | 说明 |
|----|-----|------|
| `LIDAR_BAUDRATE` | 150000 | 串口波特率 |
| `SCAN_INTERVAL_SEC` | 2.0 | 看门狗超时秒数 |
| `MAX_LSN` | 120 | 单个包最大采样点保护上限 |

---

## 协议格式 (无 CS 字段)

| 字段 | 偏移量 | 大小 | 说明 |
|------|--------|------|------|
| PH | 0-1 | 2 | 包头 0xAA 0x55 |
| CT | 2 | 1 | 控制字节，bit0=1 表示新一圈开始 |
| LSN | 3 | 1 | 采样点数量 |
| FSA | 4-5 | 2 | 起始角度 raw16 |
| LSA | 6-7 | 2 | 结束角度 raw16 |
| Samples | 8 ~ 8+LSN*3-1 | 3 each | 每个采样：距离(2) + 强度(1) |
| **总包大小** | - | **8 + LSN * 3** | - |

## BaseDriver 类 (base_driver.cpp)

### 公开方法

| 函数 | 说明 |
|------|------|
| `BaseDriver()` | 构造函数：声明参数、打开串口、订阅/cmd_vel、启动 odom 发布定时器（50ms）和 cmd 超时检查（100ms） |
| `~BaseDriver()` | 析构函数：发送停止命令、关闭串口 |

### 私有方法

| 函数 | 说明 |
|------|------|
| `bool openSerial(const std::string& port, int baudrate)` | 打开并配置 USB 串口 (termios2) |
| `void sendCmd(uint8_t cmd_byte)` | 发送 3 字节命令：AA + CMD + 55 |
| `uint8_t twistToCmd(double linear_x, double angular_z)` | 将速度指令映射为 STM32 命令：0x00前进/0x01停止/0x02后退/0x03左转/0x04右转 |
| `void cmdVelCallback(const Twist::SharedPtr msg)` | cmd_vel 回调：限幅 → 映射 → 发送串口命令 |
| `void timeoutCheckCallback()` | 每 100ms 检查：若超时未收到 cmd_vel 则自动停止 |
| `void odomTimerCallback()` | 每 50ms：航位推算积分（x,y,theta）+ 发布 /odom + TF odom→base_link |
| `geometry_msgs::msg::Quaternion quaternionFromYaw(double yaw)` | 偏航角转四元数 |

### STM32 通信协议

| 命令 | 字节流 | 功能 |
|------|--------|------|
| 前进 | AA 11 55 | 小车前进 |
| 后退 | AA 12 55 | 小车后退 |
| 停止 | AA 13 55 | 小车停止 |
| 前左转 | AA 14 55 | 小车向前左转 |
| 前右转 | AA 15 55 | 小车向前右转 |

## PestMonitor 类 (pest_monitor.cpp)

### 公开方法

| 函数 | 说明 |
|------|------|
| `PestMonitor()` | 构造函数：声明参数、订阅 /amcl_pose 和 /pest_detected、创建 publisher、启动 TCP 连接和位姿日志定时器 |
| `~PestMonitor()` | 析构函数：关闭 TCP socket |

### 私有方法

#### ROS 回调

| 函数 | 说明 |
|------|------|
| `void amclPoseCallback(const PoseWithCovarianceStamped::SharedPtr msg)` | AMCL 位姿回调：更新当前小车坐标 (x, y, theta)，发布 /robot_pose |
| `void pestDetectedCallback(const PestDetection::SharedPtr msg)` | 病虫害检测回调：结合当前 AMCL 位姿记录病虫害坐标，发布 /pest_report + RViz 标记 + TCP 上传 |
| `void tcpReconnectCallback()` | TCP 重连定时器：每 5 秒检查连接状态，断线时自动重连 |
| `void poseLogCallback()` | 位姿日志定时器：定期打印小车位置和病虫害记录数 |

#### RViz 可视化

| 函数 | 说明 |
|------|------|
| `void publishMarker(const PestRecord& record)` | 发布球体标记 + 文字标记到 /pest_markers，颜色按病虫害类型区分 |

#### TCP 局域网上传

| 函数 | 说明 |
|------|------|
| `void connectTcp()` | 创建 TCP socket 并连接到电脑端服务器 (3 秒超时) |
| `void sendTcpData(const std::string& data)` | 通过 TCP 发送 JSON 数据 (追加换行符) |

#### 工具函数

| 函数 | 说明 |
|------|------|
| `static double quaternionToYaw(const Quaternion& q)` | 四元数转偏航角 (弧度) |
| `static std::string getCurrentTimeStr()` | 获取当前时间 ISO 8601 格式字符串 |
| `static std::string buildJsonReport(const PestRecord& record)` | 构建病虫害报告 JSON 字符串 (用于 TCP 上传) |
| `static std::string escapeJson(const std::string& s)` | JSON 字符串转义 (处理引号、反斜杠等) |

### 数据结构

| 结构体 | 字段 | 说明 |
|--------|------|------|
| `PestRecord` | `pest_type` (uint8_t) | 病虫害类型编号 |
| | `pest_type_name` (string) | 病虫害类型名称 |
| | `confidence` (float) | 检测置信度 |
| | `map_x, map_y, map_theta` (double) | 小车在地图中的坐标 |
| | `timestamp` (string) | 检测时间戳 |
| | `image_path` (string) | 快照图片路径 |

### 话题接口

| 方向 | 话题 | 类型 | 说明 |
|------|------|------|------|
| 订阅 | `/amcl_pose` | PoseWithCovarianceStamped | AMCL 定位位姿 |
| 订阅 | `/pest_detected` | PestDetection | 视觉模型病虫害检测 |
| 发布 | `/pest_report` | PestReport | 病虫害报告 (含坐标) |
| 发布 | `/pest_markers` | MarkerArray | RViz 可视化标记 |
| 发布 | `/robot_pose` | PoseStamped | 小车当前位姿 (调试用) |

### 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `server_ip` | string | "192.168.1.100" | 电脑端 TCP 服务器 IP |
| `server_port` | int | 8888 | 电脑端 TCP 服务器端口 |
| `tcp_enabled` | bool | true | 是否启用 TCP 上传 |
| `log_pose_interval` | double | 5.0 | 位姿日志打印间隔 (秒) |

## TCP 上传数据格式 (JSON)

```json
{
  "type": "pest_report",
  "pest_type": 0,
  "pest_type_name": "蚜虫",
  "confidence": 0.95,
  "map_x": 1.2345,
  "map_y": 4.5678,
  "map_theta": 0.7854,
  "timestamp": "2026-06-18T10:30:00",
  "image_path": "/path/to/snapshot.jpg"
}
```

每条 JSON 数据以换行符 `\n` 结尾，电脑端 Python 可按行读取。
